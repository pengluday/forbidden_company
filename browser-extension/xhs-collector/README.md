# 小红书采集器浏览器插件

这是一个最小可用的 MV3 插件骨架，用于连接本机 Companion Service 采集小红书帖子和评论。

## 安装
1. 打开 Chrome
2. 进入 `chrome://extensions`
3. 开启开发者模式
4. 选择“加载已解压的扩展程序”
5. 选择这个目录

## 默认后端
- `http://127.0.0.1:8787`

## 交互
- 浏览器页面右下角会出现“采集本页”
- 在小红书页面右键可直接看到“采集本页帖子 + 评论”
- popup 支持采集当前页、刷新、导出 CSV / JSON
- options 页面可修改本机服务地址和默认评论上限
